[별그림자 이야기] (StarPainter)

주인공 : 별그림자 은비 (별을 그리는 자)

하늘을 돌아다니면서 지정된 위치에 별을 그리면 되는 게임

별그림자 (플레이어) : 이동 및 날갯짓(날개 이동 모션) 등
- 좌우 방향키 : 이동

별그림자 동작들 - 기력 일정량을 소모하며 기력이 다 떨어지면 땅에서 충전을 해야 한다. 날갯짓과 별 그리기는 처음부터 사용 가능하며, 도약과 순간이동은 
- 스페이스바 : 날갯짓 : 위로 점프. 공중에서도 가능
- z키 : 별 그리기 : 짧은 시간이 걸린다. 이 때는 이동할 수 없다. z키를 다시 눌러 취소한다.
- 좌우 화살표키 : 도약 : 같은 키를 '도약 입력시간' 내에 다시 입력시 가동. 해당 방향으로 재빠르게 이동한다.
- q키 : 순간이동 (관습어 '워프') : 마우스로 이동할 위치를 누른 뒤 q키를 눌러 해당 위치로 이동한다. 이 때 마우스로 누른 위치는 점선 원으로 표시된다.

게임 요소
- 체력 : 화면 아래로 떨어지거나 일부 오브젝트에 부딪히면 감소. 0 이하가 되면 스테이지 재시작
- 기력 : 플레이어 동작들에 필요. 기력이 다 떨어지면 동작 사용 불가. 기력은 땅에 있을 때 자동으로 충전된다.

기본 오브젝트
- 땅 : 고정되어 있을 수도 있고, 이동할 수도 있고, 안전하지 않은 땅으로 바뀔 수도 있다.
- 목표 지점 : 별을 그려야 하는 지점. 순서가 정해져 있거나 자유. 모든 지점에 별을 그리면 스테이지 클리어

방해물 : 방해물에 부딪히면 체력이 깎인다. 짧은 '무적 시간'이 주어진다. '무적 시간' 동안에는 방해물의 방해를 받지 않으며 이 때 플레이어는 파란색으로 표시된다. '무적 시간'을 이용하여 오브젝트의 범위에서 벗어나야 한다. 
- 혜성 : 위에서 임의의 속도로 날아오는 방해물
- 행성, 위성 : 특정 위치에 있는 방해물 / 행성의 주위를 도는 방해물
- 블랙홀 : 범위 안으로 천천히 빨려 들어가며 본체에 닿으면 체력이 깎인다. 이 때 빨려 들어가는 속도는 '별그림자'의 이동속도보다 느리기에 탈출할 수 있다.
- '사건의 지평선' 단원들 : 별의 힘을 흡수하여 우주를 암흑으로 물들이려는 세력들. 플레이어를 방해한다. 각자 이동하며 별그림자가 그리다 만 별이 있다면 그 방향으로 우선적으로 이동한다.

스테이지 화면
- 오른쪽 상단 미니맵, 좌측 하단 동작 사용 가능여부(쿨타임 여부) 표시
- 체력, 기력은 바 + 숫자 형태로 표시. 남은 별 갯수 등 표시
- 보호성 (쉴드)은 별그림자 옆에 표시된다.
- [사건의 지평선]이 일정량의 초신성 에너지를 모으면 임무 실패로 판정한다.

스테이지 중간에서는 그 동안 모은 '별의 기억'을 가지고 [별그림자 회랑]에서 '능력 전이'(강화)를 진행할 수 있다. '능력 전이'를 초기화하고 '별의 기억'을 (도약과 순간이동 해제시 드는 '별의 기억' 제외) 전부 되돌려받아 재설정하는 것도 가능하다.
:: 설정상 '능력 전이'는 [별그림자 회랑]의 수호자인 [성청화]가 수행한다.
- 처음에는 '도약'과 '순간이동'이 잠긴 상태로, 초반에 '별의 기억'을 가지고 동작을 배운다. 
- 이동속도 증가 / 체력 증가 / 기력 회복속도 증가 (최대 기력의 %단위)
- 별 그리기 : 그리기 시간 감소 / 필요 기력 감소 (최대 강화시 기력 소모 0)
- 날갯짓, 도약, 순간이동 : 필요 기력 감소 (최대 강화시에도 약간의 기력 필요)
- 보호성 (관습어 '쉴드') : 별그림자의 옆에 있으며 (최대 2체) 방해물 충돌시 별그림자를 보호하며 하나씩 사라진다. 보호성은 스테이지 시작시 보유 개수만큼 생겨난다 (보유 개수는 영구 강화에 따름).
